Description:
This program implements an application class named LizetteCalculator.
The program prompts the user to enter two integer values using keyboard input.
It multiplies the two integers and displays both the entered values and their multiplication result.

The program uses the Scanner class for input and performs basic arithmetic operations.
Clear prompts and output messages are provided to guide the user through the program execution.

Files Included:
1. LizetteCalculator.java
2. LizetteCalculator.class
3. README.txt

Instructions:
To run: Java LizetteCalculator

To test:
Enter 4 for the first integer and 2 for the second integer. The expected output is:
You entered 4 and 2. The product of the two numbers is 8.
